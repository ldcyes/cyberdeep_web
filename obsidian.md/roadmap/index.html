Roadmap

We're chipping away at improvements to Obsidian.
For more detail about individual releases see the Changelog. Have a feature request? Visit the official forum.

Active

  • Mobile startup time

    Speed improvements for opening and editing notes on mobile.

  • Multiplayer

    Share notes and edit them collaboratively.

  • Sync onboarding

    Improve onboarding for Sync, especially on mobile.

  • Web clipper

    A browser extension that helps you save content from the web.

Planned

  • Canvas support for Publish

  • Dynamic views

    Create dynamic tables using data stored in note properties.

  • Mobile quick capture

  • PDF annotation

    Currently waiting for native support in PDF.js.

  • Sort search results by relevance
